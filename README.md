# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/5cbc7ff1-424b-4f0d-a2c9-e63bad22a733

## Run Locally

**Prerequisites:** Node.js, optional MongoDB, Groq API key

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and set `GROQ_API_KEY`
3. Start the AI Core + frontend:
   - `npm run dev:all` (recommended)
   - Or separately: `npm run dev:server` + `npm run dev`
4. Open http://localhost:3000

See [server/README.md](server/README.md) for API docs and architecture.
